### Constraints
- Github privado [Gitflow]
- Styled components
- Typescript

- Express / NestJs / Sequelize / Type ORM

- Jest / Testing library / Enzyme / Cypress 
- Readme (Por favor anota todos los concern que tuviste durante el desarrollo y que decisión tomaste al respecto.)
